Summary: Latitude vs. Weather Systems

Using the OpenWeatherMap API, I pulled weather data from over 600 cities around the world. The purpose of this assignment was to compare the latitudes of cities and weather patterns. In the project we looked at humidity, cloudiness, maximum temperature, and windspeed. The first step was to randomly generate latitudes and longitudes to create a list of 600+ cities. Then the API was used to pull the weather data at those locations.

Once I had the data I pulled it into a table and then applied additional filters. With the data in a table, I used matplotlib to chart the information for the various factors that we were interested in exploring. For each of the four factors (humidity, cloudiness, maximum temperature, and windspeed) I created scatter plots with the latitude along the x-axis and the different factors on the y-axis. Each dot on the scatter plot represents a city at their location and the corresponding weather data taken that day.
